1. Create a database called "vitl_demo "


2. Create a table within the database using the script below


	CREATE TABLE names (
    name_id int NOT NULL AUTO_INCREMENT,
    first_name varchar(100) NOT NULL,
    last_name varchar(100),
      PRIMARY KEY (name_id)
);


3. Import "vitl_demo.sql" into the "names" Table

4. Edit "Config.php" with your database credentials

5. Run the script.

NB: Please type first 3 letters of the last name  into the search box and strike the spacebar on your keyboard.
